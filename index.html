nostr-post-checker

イベントがどのリレーに存在し、どのリレーに存在していないかを確認するサイトです。
You can check which relay servers the event exists on and which do not.


Get by NIP-07 and kind:3 content
Get by NIP-07 and kind:10002 r tags

result:

使い方:

  • event id ボックスに調べたい"note1～", "nevent1～", "npub1～", "nprofile1～" を入力します。
  • kind ボックスに調べたいkindの数字を入力します。
  • relays ボックスに調べたいリレーの URL を入力します。複数リレーを調べたいときは改行で追加してください。

usage:

  • Put "note1...", "nevent1...", "npub1..." or "nprofile1..." you want to check into the "event id" box.
  • Put the kind number you want to check in the "kind" box.
  • Put the relay addresses you want to check in the "relays" box.
